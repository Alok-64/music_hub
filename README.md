# music_hub
A dark-themed music search and download app powered by Spotify API, featuring instant MP3 downloads and a sleek glassmorphism UI.

Music Hub is a sleek, dark-themed music discovery platform that brings the power of Spotify's catalog to your fingertips. Search millions of tracks, preview them with Spotify's embedded player, and download full MP3 files instantly - all without a backend server.

Built entirely with vanilla HTML, CSS, and JavaScript, this project showcases modern web design with glassmorphism effects, smooth animations, and a fully responsive layout that works seamlessly across all devices. The app leverages the Spotify Downloader API via RapidAPI to provide search, playback, and download functionality directly from the browser.
